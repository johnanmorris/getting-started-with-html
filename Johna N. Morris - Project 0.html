The Basics of the Web

The Web

The web is essentially a collection of documents. Most of these documents are HTML documents (but there are also PDFs, image files, etc). A user uses a web browser on his or her computer to view the Internet. In order to do this, the user's computer must send a request with HTTP (hypertext transfer protocol) to servers which store the HTML pages. The Internet is just a collection of many servers connected together, interpreted through a web browser.

HTML stands for HyperText Markup Language. An HTML document has three basic components:
text, or the main content of a page;
markup, the code telling the computer what a page should look like; and
links to other pages.

HTML Markup

The basic unit of HTML markup is the tag. A tag is always surrounded by angle brackets (< >). Most tags have an opening tag and a closing tag. A closing tag has a forward slash after the first bracket (). Everything between tags is known as an element.

Some tags have what is known as an attribute. An attribute is a property of an HTML element. For example, the anchor tag, used to create links (like this one to Google), has an attribute known as href. Tags can have more than one attribute; for instance, the img tag, used for linking to and displaying images, usually has two attributes, src (for source) and alt (for alternate). The src tag is used to link to the url of the image, while alt displays text if a link to the image is broken or a page is being viewed by a blind person.

img tags are a tag type known as a void tag. This simply means that there is no content (text) within the tag. Void tags do not need closing tags (and, in newer HTML, are "self-closed" with a forward slash immediately before the final angle bracket, like < />). Another void tag is used for line breaks, br.

Inline Tags vs. Block Tags

Inline tags are tags which occur within a text. Many common tags are inline tags, such as:
line break: br ;
image: img ;
italics: em ; and
bold: b or strong.

On the other hand, a block tag, is one that sections off an entire section of text, creaking an invisible "box" around an element. The most common block tag is probably the paragraph, or p tag.

Span and Div

The difference between inline and block tags can be illustrated by comparing the span and div tags. Both tags are container tags—they surround an element in a certain style. However, span is an inline tag, existing within a line, while div is a block tag, creating a "box" or block around an element.

HTML Document Structure

There are many tags, but all HTML pages have the following parts which make up the backbone of a page:

!DOCTYPE HTML : tells you what the document type is
html : This tag starts and ends each HTML document
head : The head tag contains the metadata of a page. This can include the title (enclosed with title tags), links to any referenced CSS or JavaScript pages, and the like.
body : These tags enclose the main body of the page—all the text, images, and links within a page.
